📊 Googleスプレッドシート連携

2通りの方法でスプレッドシートに記録を反映できます。

方法①：CSVをインポート（簡単）
  1. 「CSVダウンロード」でファイルを保存
  2. Googleスプレッドシートを開く
  3. 「ファイル → インポート → アップロード」でCSVを選択
方法②：Apps Script自動連携
  1. Googleスプレッドシートを新規作成
  2. 「拡張機能 → Apps Script」を開く
  3. 下のコードを貼り付けて保存
  4. 「デプロイ → 新しいデプロイ」→ 種類:ウェブアプリ → アクセス:全員 → デプロイ
  5. 表示されたURLを下の欄に貼り付け
